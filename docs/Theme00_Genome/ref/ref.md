# 注释信息

`ENSEMBL` 与 `SYMBOL` 在不同版本中的对应关系不一样。

一般来说，【这样理解对吗？】
- `ENSEMBL` 基本不会变，它表述的是基因在基因组上的位置关系。
- `SYMBOL` 可以理解为在这段区域上的注释信息，可随着研究深入更迭，不断发生变化。
